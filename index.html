Documentation Cryptographique

1. Chiffrement Symétrique (AES)

AES-256 - Algorithme rapide pour gros volumes de données.

2. Chiffrement Asymétrique (RSA)

RSA-2048 - Distribution facile des clés, idéal pour échange.

3. Chiffrement Hybride

Combine RSA + AES : message chiffré avec AES, clé AES chiffrée avec RSA.

4. Principe de Kerckhoffs

La sécurité repose uniquement sur la clé.
